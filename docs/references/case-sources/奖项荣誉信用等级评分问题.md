# 奖项荣誉信用等级评分问题

## 官方来源

- 来源网站：中国政府采购网
- 官方链接：[原文链接](https://www.ccgp.gov.cn/llsw/202508/t20250827_25238222.htm)

## 审查中最有价值的口径

### 1. 与履约无关的奖项、荣誉、信用等级不宜作为重要评分因素

重点口径：
- 将重合同守信用单位、知名品牌企业、优秀供应商、AAA级信用证书等作为评分因素，容易与履约能力脱节
- 若与采购标的质量或服务效果没有直接关系，容易形成不合理差别待遇

常见适用场景：
- AAA信用等级加分
- 省级及以上荣誉称号加分
- 高新技术企业、3A/5A认证、品牌荣誉等评分

### 2. 奖项和资质类评分需审查与采购标的的直接关联

重点口径：
- 不应简单把供应商综合形象、市场声誉当作履约能力替代物
- 应回到项目实际需求和可验证质量指标

常见适用场景：
- 服务类采购中对品牌荣誉、专利、企业称号高分加权
- 将行业荣誉直接作为资格门槛或核心评分项

结合中国政府采购网关于采购需求编制常见问题的实务分析，与采购标的质量或直接履约能力无明显关系的奖项、荣誉、信用等级、企业称号，不宜设置为重要评分因素或资格门槛。当前条款存在以综合声誉替代履约能力评价的倾向。
