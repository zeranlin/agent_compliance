## 审查中最有价值的口径

### 1. 地域限制属于高敏感问题

重点口径：
- 招标文件中对供应商设置地域限制，通常属于政府采购中的高风险问题
- 审查重点应放在是否符合非歧视性要求、是否确有项目必要性

常见适用场景：
- 要求本地分公司、本地服务网点、本地备案
- 要求特定行政区域内的业绩、纳税记录或办公场所

### 2. 非歧视性审查要关注资格条件、技术要求和评审因素

重点口径：
- 资格条件设置是否合理
- 是否要求提供过多同类业务合同
- 技术要求是否指向特定供应商、品牌、技术路线
- 评审因素设置是否具有倾向性

常见适用场景：
- 同类业务合同要求过多
- 地域、行业、金额定向业绩
- 本地服务便利性被泛化成准入门槛或高权重评分项
