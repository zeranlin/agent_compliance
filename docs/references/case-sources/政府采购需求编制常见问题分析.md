# 政府采购需求编制常见问题分析

## 官方来源

- 来源网站：中国政府采购网
- 官方链接：[原文链接](https://www.ccgp.gov.cn/llsw/202508/t20250827_25238222.htm)

## 审查中最有价值的口径

### 1. 不应设置与采购标的或履约无关的门槛和加分项

重点口径：
- 人员、资质、荣誉、奖项、专利、信用等级等，如果与标的履约无直接关系，不宜设置为不合理门槛或高权重评分因素

常见适用场景：
- 奖项作为资格条件
- AAA信用等级加分
- 荣誉称号、认证证书、专利作为重要评分项

### 2. 采购需求和评分标准应围绕项目实际需要

重点口径：
- 采购需求不应脱离项目实际功能
- 评分规则不应人为偏向少数供应商的组织特征或综合资源

常见适用场景：
- 把增值服务能力纳入核心评分
- 把与项目核心履约关联弱的综合平台能力设为高分项
