# 资格条件与类似业绩设置

## 官方来源

- 来源网站：中国政府采购网
- 官方链接：[原文链接](https://www.ccgp.gov.cn/llsw/202405/t20240524_22184337.htm)

## 审查中最有价值的口径

### 1. 资格条件应与采购标的功能、质量和履约能力直接相关

重点口径：
- 资格条件应围绕采购标的本身设定
- 应属于履行合同所必需的条件
- 不应泛化设置与标的关联弱的资格门槛

常见适用场景：
- 强制要求大量专业证书
- 强制要求与履约无明显关系的人员配置
- 以组织规模、设立年限替代履约能力

### 2. 类似业绩可以设置，但应控制范围和强度

重点口径：
- 类似业绩要求应与项目核心标的一致
- 不应要求过多同类合同
- 不应设置特定行业、特定金额、特定区域的定向业绩

常见适用场景：
- 同类合同数量要求过高
- 指定必须为本地或特定行业业绩
- 把相邻但并非核心的服务全部并入类似业绩范围

## 可直接复用的引用提示

```text
结合中国政府采购网关于类似业绩设置的实务分析，资格条件应与采购标的功能、质量及履约能力直接相关，类似业绩要求也应围绕项目核心标的设定。当前条款对资格或类似业绩的要求过宽/过高，存在缩小竞争范围的风险。
```

## 使用提醒

- 该来源适合判断资格条件是否泛化、类似业绩是否过宽或过高。
- 使用时建议同时搭配《政府采购需求管理办法》和《政府采购法实施条例》。
