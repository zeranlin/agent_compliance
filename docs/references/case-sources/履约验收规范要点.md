## 审查中最有价值的口径

### 1. 验收方案应以采购文件、投标响应和合同约定为基础

重点口径：
- 验收清单和标准应从采购文件、投标文件、合同文本中逐一对应形成
- 验收不能脱离采购需求临时增加或模糊化

常见适用场景：
- 采购文件未写清验收标准
- 中标后再以演示、样品、满意度等主观标准决定是否验收

### 2. 验收过程应逐项核对并形成记录

重点口径：
- 应根据履约验收方案逐一核对
- 应做好验收记录并出具验收意见

常见适用场景：
- 只写“动态考核”“综合评价”
- 没有验收记录、验收流程和责任主体
