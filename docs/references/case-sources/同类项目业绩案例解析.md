## 审查中最有价值的口径

### 1. 同类项目业绩范围应与采购标的保持一致

重点口径：
- “同类项目业绩”应当围绕采购标的核心内容来定义
- 不宜无限扩大到相邻但并非核心的业务类型

常见适用场景：
- 代理记账项目将融资辅导、培训、并购重组等也算作同类业绩
- 软件项目把咨询、运维、培训等全部混同为同类项目

### 2. 同类业绩表述应明确、稳定、便于评审

重点口径：
- 业绩定义不清容易导致评审标准不统一
- 也容易引发质疑投诉

常见适用场景：
- 业绩范围写得过宽
- 业绩范围写得模糊
- 要求提供过多附加证明，实质上提高了进入门槛

结合中国政府采购网关于“同类项目业绩”争议的案例解析，同类项目业绩应围绕采购标的核心内容设定，并做到定义明确、评审口径统一。当前条款将与核心标的关联较弱的服务一并纳入同类业绩，存在扩大倾斜范围的风险。
