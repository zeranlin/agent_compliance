## 审查中最有价值的口径

### 1. 主观评审因素应细化、量化、分级分档

重点口径：
- 主观评分设置细化量化程度低，会直接影响评审公平
- 应将主观评审因素分级分档，并把对应分值写明确
- 应尽量减少主观分设置

常见适用场景：
- 工作方案、现场陈述、平台演示采用“综合比较”打分
- 使用“优、良、中、差”“针对性强、可操作性强”等模糊描述

### 2. 主观评分应留痕并可复核

重点口径：
- 对主观评分差异过大的情况，应能说明原因
- 高分、低分都应有明确评价依据

常见适用场景：
- 专家评分差异巨大但缺少理由
- 文件只给主观分值，不给评分理由框架
