# 综合评分法边界分析

## 官方来源

- 来源网站：中国政府采购网
- 官方链接：[原文链接](https://www.ccgp.gov.cn/llsw/202511/t20251112_25678095.htm)

## 审查中最有价值的口径

### 1. 评分项应是与采购需求对应的量化指标

重点口径：
- 参与评分的指标应与采购需求对应，且应量化
- 客观但不可量化的指标，应作为实质性要求，而不是评分项

常见适用场景：
- 平台能力作为演示打分，但实际上无法量化
- 服务方案、实施方案、界面体验等难以客观量化内容被高权重计分

### 2. 资格审查和符合性审查内容不得再作为评分项

重点口径：
- 资格审查证明材料、符合性审查的实质性需求指标，不应再作为评分项
- 与采购需求无直接关联的因素，也不得作为评分项

常见适用场景：
- 同类业绩、人员资质、专业能力既作为资格条件又作为重要分值项
- 把本地纳税额、奖项荣誉、信用等级等作为评分项

## 可直接复用的引用提示

```text
结合中国政府采购网关于综合评分法边界的实务分析，资格审查材料、符合性审查中的实质性需求指标，以及客观但不可量化的内容，不宜再作为评分项。当前条款存在将资格或实质性要求重复转化为评分优势的风险。
```

## 使用提醒

- 该来源特别适合判断“资格条件是否被重复计分”“实质性要求是否被再次打分”。
- 作为实务补强使用时，应搭配《政府采购需求管理办法》或评审规则类正式依据。
