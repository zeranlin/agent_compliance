# 合同付款前提与责任边界

## 官方来源

- 来源网站：中国政府采购网
- 官方链接：[原文链接](https://www.ccgp.gov.cn/llsw/202506/t20250606_24726813.htm)

## 审查中最有价值的口径

### 1. 不宜设置不合理付款前提条款

重点口径：
- 将第三方付款作为向中小企业付款前提，存在较高合同风险
- 付款条款应围绕采购人与供应商之间的合同关系明确约定

常见适用场景：
- 约定“待上级拨款后支付”
- 约定“待第三方付款后支付”
- 支付条件与供应商可控履约行为无直接关系

### 2. 合同责任边界应明确，不能以绝对免责替代责任分配

重点口径：
- 合同应明确责任主体和风险承担方式
- 不宜使用“甲方不承担任何责任”“一切责任均由乙方承担”等绝对化表述

常见适用场景：
- 采购人绝对免责
- 数据、合规、连带责任全部转嫁
- 责任条款与采购文件、合同文本不闭环

## 可直接复用的引用提示

```text
结合中国政府采购网关于新增采购风险的实务分析，合同付款条件和责任边界应与采购合同关系相匹配，不宜设置与第三方付款挂钩的前提条款，也不宜以绝对免责方式排除法定或约定义务。当前条款存在付款条件不清/责任失衡风险。
```
