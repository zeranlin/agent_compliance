# 案例口径本地引用库

## 目的

本目录用于保存可直接支持审查判断的典型案例、实务分析和监管口径摘要。

## 使用规则

- 案例口径用于补强实务判断，不替代法律和部门规章。
- 审查输出时，优先引用高位规则，再补充案例口径。
- 案例如时间较久，应注意复核是否仍具有参考价值。
