# 法规依据本地引用库

## 目的

本目录用于保存政府采购需求合规性审查中高频使用的官方法规依据摘要，供后续智能体和人工审查直接检索与引用。

## 使用规则

- 这里只存“适合直接引用的摘要”和“适用场景”，不替代原始官方文件。
- 每个文件都应保留原始官方链接。
- 如规则可能发生变化，应优先复核官方来源后再使用。
- 对“财政部规章”类文件，建议同时记录权威目录页、令号、颁布日期和有效性状态，不要只保留正文链接。
- 对“通知”“办法”“办公厅文件”等非规章资料，不要误标为“财政部规章”；应单独说明核验来源。
