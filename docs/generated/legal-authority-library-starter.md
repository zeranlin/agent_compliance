# 法规依据库样表

## 说明

本表用于沉淀政府采购需求合规性审查中最常用、最稳定的法规和政策依据。

使用方式：
- 每条规则对应一个唯一编号。
- 审查输出时，优先引用本表中已经整理好的规则摘要。
- 如规则存在失效或替代情况，应在“状态”和“备注”字段中更新。

## 字段定义

- `rule_id`：规则编号
- `层级`：法律、行政法规、部门规章、规范性文件、地方规则、案例口径
- `文件名称`：正式文件名
- `发文机关`：发布机关
- `适用主题`：资格条件、评分标准、技术要求、合同条款、验收、框架协议等
- `核心约束`：适合直接用于审查的简明规则
- `适用场景`：何时优先引用
- `引用提示`：输出审查意见时可复用的表达方向
- `状态`：有效、待核实、地方适用
- `备注`：生效、替代、适用边界等说明
