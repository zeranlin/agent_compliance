# [BACG2025000096-A]深圳市宝安区中心医院胃肠镜类设备采购项目 审查意见

## 审查信息

- 审查对象：`[BACG2025000096-A]深圳市宝安区中心医院胃肠镜类设备采购项目.docx`
- 审查时间：`2026-03-16`
- 审查范围：资格条件、评分规则、技术要求、商务及验收条款
- 审查结论：当前文件存在多处高风险条款，尤其集中在供应商资格条件、评分项设置和商务条款失衡方面。按现有文本直接发标，存在被质疑、投诉或被认定为差别待遇、评审因素不当或合同责任失衡的较高风险。

## 审查依据

- 《中华人民共和国政府采购法实施条例》第二十条关于不得以不合理条件对供应商实行差别待遇或者歧视待遇的要求
- 《政府采购需求管理办法》关于采购需求、评审因素、合同条款和履约验收应围绕项目特点与实际需要设置的要求
- 中国政府采购网关于非歧视性审查、综合评分法边界、奖项荣誉评分、类似业绩设置、履约验收与付款责任边界的实务口径
- 本地引用资料库中已核验的法规依据和案例口径

## 总体风险摘要

1. 资格条件中出现了所有制、股权结构、经营年限、财务指标和企业称号等直接限制，属于最突出的合规风险。
2. 评分标准中混入与项目直接履约关系较弱的业绩、荣誉、企业证明和营业执照加分，同时主观评分表述较多，评审自由裁量空间偏大。
3. 技术参数中存在若干疑似指向特定技术路线或产品体系的表述，部分参数组合需要采购人补充市场调研和临床必要性说明。
4. 商务和验收条款中存在与项目标的不直接相关的义务，以及付款、第三方检测、信息系统对接和绿色义务明显向供应商单方倾斜的情形。

### 4. 主要问题：售后服务评分项混入无关业绩、荣誉和证照加分
- 位置：评标信息-技术部分-投标人承诺的售后服务内容和工作方案
- 页码提示：Word 原件待人工翻页复核
- 条款编号：评分因素第5项
- 辅助行号：0319-0322
- 原文摘录：`如果在医疗行业存在相关相关项目的业绩，可得10分。若供应商提供为守合同重信用企业，可得10分。若供应商提供全国科技型中小企业证明，可以得10分。若供应商提供营业执照或事业单位法人证书等证明资料扫描件，可得1分。`
- 风险类型：`duplicative_scoring_advantage`
- 风险等级：高
- 合规判断：`likely_non_compliant`
- 依据：
  - 《政府采购需求管理办法》要求评审因素尽可能客观、量化，并与采购需求相对应。
  - 中国政府采购网关于综合评分法边界和奖项荣誉评分问题的实务口径均指出，资格证明材料和与履约无关的荣誉称号不宜作为评分项。
- 适用逻辑：该评分项名为“售后服务内容和工作方案”，但实际混入类似业绩、守合同重信用企业、科技型中小企业和营业执照加分。其中营业执照本属资格证明，不应再转化为得分优势；企业荣誉和企业称号与售后方案质量也不直接对应。
- 修改建议：删除上述附加得分点，仅保留售后团队配置、响应时限、备件保障、培训方案等与售后履约直接相关且可量化的指标。
- 建议替代表述：`根据售后服务方案中的响应时效、维保组织、备件供应、培训安排和支持机制量化评分，不再对企业荣誉、营业执照或企业称号单独加分。`

- 风险类型：`other`

  - 中国政府采购网关于采购需求编制常见问题的实务口径提示，需求不得脱离项目实际功能。

### 8. 主要问题：付款条款和风险分配明显向采购人单方倾斜
- 位置：第三章 用户需求书-五、商务要求-交货及付款要求
- 页码提示：Word 原件待人工翻页复核
- 条款编号：3.2 付款方式
- 辅助行号：1536-1550
- 原文摘录：`若因为财政审批的原因造成采购人延期付款的，采购人不承担违约责任……若本项目财政下达资金数不足以支付合同约定比例的金额，则采购人按本项目财政下达资金数支付合同款……预付款未支付时，中标人仍必须在交货期内交货。`
- 风险类型：`one_sided_commercial_term`
- 风险等级：高
- 合规判断：`potentially_problematic`
- 依据：
  - 《政府采购需求管理办法》要求合同权利义务应围绕采购需求和合同履行设置，并充分考虑风险分担。
  - 中国政府采购网关于合同付款前提与责任边界的实务口径提示，不宜设置与供应商不可控因素挂钩的付款前提，也不宜通过绝对免责方式排除应承担的责任。
- 适用逻辑：财政支付延迟风险、资金不足风险和预付款未到账风险几乎全部转由供应商承担，且采购人迟延付款基本被免责，权利义务失衡较明显。
- 修改建议：对财政支付程序导致的客观延迟可约定合理顺延，但应同步约定说明、协商和补付安排，不宜当然免除全部责任；删除“预付款未支付仍须交货”等明显失衡表述。
- 建议替代表述：`因财政支付程序导致付款延迟的，采购人应及时书面说明原因并与中标人协商新的支付节点；对超出合理期限的延迟，双方按合同约定另行处理。`

### 9. 主要问题：验收和第三方检测条款过于宽泛，且检测费用全部由中标人承担
- 位置：第三章 用户需求书-五、商务要求-验收条件
- 页码提示：Word 原件待人工翻页复核
- 条款编号：4.3-4.6
- 辅助行号：1609-1631
- 原文摘录：`产品质量和安装调试检验标准遵照国家相关规定和最新标准执行。` `如采购人需要，中标人需无条件配合采购人委托有资质的第三方质量检测部门按照招投标文件进行技术参数检测确认，委托检测验收事项由中标人负责。`
- 风险类型：`unclear_acceptance_standard`
- 风险等级：中
- 合规判断：`potentially_problematic`
- 依据：
  - 《政府采购需求管理办法》要求验收方案明确主体、时间、方式、程序、内容和验收标准。
  - 中国政府采购网关于履约验收规范要点的实务口径提示，验收标准应以采购文件、投标响应和合同约定为基础，形成逐项核对清单。
- 适用逻辑：仅以“国家相关规定和最新标准”概括验收，缺少对应本项目关键技术与商务要求的逐项验收清单；同时“如采购人需要”即可启动第三方检测且费用概由中标人承担，边界不清。
- 修改建议：把核心技术参数、安装调试、系统对接、培训和维保资料等逐项转化为验收清单；明确第三方检测触发条件、检测范围、费用承担规则和争议处理机制。
- 建议替代表述：`双方按合同附件中的履约验收清单逐项验收。仅在对关键技术指标存在合理争议时，可委托有资质的第三方检测；检测项目、标准和费用承担规则应事先明确。`

  - 《政府采购需求管理办法》要求采购需求和合同权利义务围绕项目目标设置，并审查绿色采购要求是否必要、是否与项目相适应。
- 适用逻辑：绿色采购政策可以落实，但本条义务较重，且未说明报告频次、标准口径、适用对象和与设备供货履约的直接关系，容易增加不确定成本。
- 修改建议：如采购人确需落实绿色采购，建议改为符合现行绿色包装、节能环保产品目录或可提供通行环境声明材料的要求，不宜附加持续监控型义务；如坚持保留，应补充适用依据和交付标准。
- 建议替代表述：`如适用绿色采购政策，投标人按现行政府采购绿色产品、包装和节能环保要求提供相应承诺或证明材料。`

## 需要人工采购/法务复核的事项

1. 技术参数中关于兼容性、无线插拔、防水盖、探头频率和外径等要求，是否已有充分市场调研和临床必要性论证。
2. 拒绝进口产品是否已完成适用政策和国产替代充分性论证，尤其对超声内镜相关设备是否已有专项说明。
3. 5年整机原厂维保、95%开机率及低于85%必须更换新设备等维保条款，是否符合市场惯例和预算安排。
4. 免费开放端口并无偿完成与HIS、PACS、LIS及未来变更系统的数据对接义务，是否与信息化边界、接口标准和预算安排一致。
