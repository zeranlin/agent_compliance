# [BACG2025000089-A]深圳市宝安区中心医院血透类设备采购项目 人工式审查意见

## 一、总体结论

总体结论：`高风险，不建议按现文本直接发标。`

本项目采购标的为血透类设备，但现有文本同时存在资格条件错位、评分主题漂移、技术证明形式过严、商务和验收责任整体偏重、以及与医院信息化和碳足迹相关的附加义务外扩等问题。若按现文本直接发标，容易引发针对资格、评分和商务条款的质疑投诉。

## 二、主要问题

### 1. 资格条件中混入与标的不匹配的资质和企业条件

- 位置：第一章 招标公告-申请人的资格要求
- 页码提示：第13页左右（Word 原件建议人工翻页复核）
- 条款编号：资格要求第11-14项
- 辅助行号：548-551
- 原文摘录：
  - `农民专业合作社不具备投标资格。`
  - `投标人须具备高空清洗悬吊作业企业安全生产证书`
  - `投标人须具备高新技术企业证书。`
  - `投标人须具备《企业诚信管理体系认证证书》。`
- 风险类型：`qualification_domain_mismatch`
- 风险等级：高
- 合规判断：`likely_non_compliant`
- 依据：
  - 《中华人民共和国政府采购法实施条例》第二十条
  - 《政府采购需求管理办法》关于重点审查资格条件设置合理性的要求
- 适用逻辑：血透类设备供货项目的资格条件应围绕法定主体资格、医疗器械生产经营许可和基本履约能力设置。将`高空清洗悬吊作业企业安全生产证书`、`高新技术企业证书`、`企业诚信管理体系认证证书`写入资格门槛，与血透设备供货履约没有直接对应关系，且排除`农民专业合作社`也缺少合法基础。
- 修改建议：删除与设备供货履约无直接关系的资质、称号、管理体系和组织类型限制，仅保留医疗器械法定许可和主体资格要求。
- 建议替代表述：`投标人应具备独立承担民事责任的能力，并具备所投医疗器械依法生产或经营所需的许可、备案或注册条件。`

### 2. 评分项名称、评分内容与评分证据明显错位

- 位置：评标信息-商务部分
- 页码提示：第5-10页左右
- 辅助行号：222-225，358-423
- 原文摘录：
  - `投标人提供符合相关要求的承诺函得100分`
  - `投标人具有建设部门（或事业单位）颁发的“绿色物业管理评价标识”荣誉证书得30分`
  - `投标人的注册资本500万以上加20分`
  - `年均营业收入不低于500万元，得20分`
  - `ISO9001...得60分；ISO13485...得40分`
- 风险类型：`scoring_content_mismatch`
- 风险等级：高
- 合规判断：`likely_non_compliant`
- 依据：
  - 《政府采购需求管理办法》
  - 财政部关于综合评分法边界和主观评审客观化的监管口径
- 适用逻辑：绿色采购承诺、物业荣誉、注册资本、营收净利润、体系认证等内容被并入多个评分主题，导致评分项名称、计分内容和证明材料之间明显不一致，并把一般经营指标和与设备履约弱相关的证书转化为高分优势。
- 修改建议：按评分主题重构评分表，删除与履约弱相关的荣誉、资本、营收、净利润加分；体系认证只可少量辅助评价，不宜设为高分决定项。

### 3. 同类业绩和认证评分权重过高，显著放大既有供应商优势

- 位置：评标信息-商务部分
- 页码提示：第8-10页左右
- 辅助行号：357-423
- 原文摘录：
  - `血透类设备项目供货或销售业绩...每提供一项得40分`
  - `以上4项累积，本项最高得100分`
  - `ISO9001...得60分；ISO13485...得40分`
- 风险类型：`excessive_scoring_weight`
- 风险等级：高
- 合规判断：`likely_non_compliant`
- 依据：
  - 《政府采购需求管理办法》
  - 中国政府采购网关于同类业绩和认证评分的典型争议口径
- 适用逻辑：将同类业绩和体系认证分别抬到`100分`量级，会显著放大既有供应商和成熟厂商的竞争优势，使评审结果被少数证明材料决定。
- 修改建议：压降单类因素分值上限，将高分项拆散为与本项目安装调试、售后保障、产品性能直接相关的多个可验证指标。

### 4. 售后响应、到场和备用机要求形成事实上的本地化倾向

- 位置：第三章 用户需求书-★五、商务要求
- 页码提示：第36页左右
- 条款编号：1.1、1.1（保外）
- 辅助行号：1577，1605
- 原文摘录：`2小时内响应，12小时内到达现场维修，48小时内消除故障...10个日历天内免费提供备用设备`
- 风险类型：`geographic_restriction`
- 风险等级：高
- 合规判断：`potentially_problematic`
- 依据：
  - 《中华人民共和国政府采购法实施条例》第二十条
  - 《政府采购需求管理办法》
- 适用逻辑：血透设备确有售后响应需求，但`2小时响应+12小时到场+48小时排障+10天内免费备用机`的组合要求，会对本地驻点或既有服务网络形成明显优势，若无充分运维场景论证，属于事实上的本地化倾斜。
- 修改建议：改为按故障等级区分响应机制，明确电话响应、远程支持、现场到场和备用机触发条件，不直接用极短到场时限替代履约能力。

### 5. 技术参数和证明形式要求过严，存在定向和取证门槛叠加风险

- 位置：第三章 用户需求书-招标技术要求
- 页码提示：第19页以后
- 辅助行号：821-920
- 原文摘录：
  - `依据EN14175-3:2019标准`
  - `检验检测内容符合ISO 20743抗菌+ISO 10993`
  - `须提供CMA检测报告及全国认证认可信息公共服务平台截图`
  - `机身尺寸...强制要求供应商提供第三方机构出具的专项检测报告`
- 风险类型：`narrow_technical_parameter` / `technical_justification_needed`
- 风险等级：高
- 合规判断：`needs_human_review`
- 依据：
  - 《政府采购需求管理办法》
  - 中国政府采购网关于技术参数设置与检测证明要求的争议口径
- 适用逻辑：血透设备技术参数中叠加了与设备场景关联不明的标准引用、专项检测、CMA报告和平台截图要求，容易形成“技术要求 + 证明形式”双重门槛。部分要求可能需要临床或院感论证，但当前文本没有给出必要性依据。
- 修改建议：保留真正影响临床使用的核心参数，删除与标的不匹配或必要性不明的标准引用；证明方式优先采用产品说明书、注册资料、出厂检测或等效证明，不宜普遍强制第三方检测截图。

### 6. 验收、检测、付款和责任条款整体偏向供应商承担

- 位置：第三章 用户需求书-商务要求、验收条件；第五章 合同条款
- 页码提示：第37-42页、第70页以后
- 辅助行号：1580-1881，3272以后
- 原文摘录：
  - `全部费用由中标供应商承担`
  - `采购人不承担违约责任`
  - `采购人有权单方解除合同`
  - `委托有资质的第三方质量检测部门...由中标人负责`
  - `预付款未支付时，中标人仍必须在交货期内交货`
- 风险类型：`one_sided_commercial_term` / `unclear_acceptance_standard`
- 风险等级：高
- 合规判断：`likely_non_compliant`
- 依据：
  - 《中华人民共和国民法典》
  - 《政府采购需求管理办法》
  - 履约验收规范相关口径
- 适用逻辑：付款、验收、检测、到场响应、备用机、费用承担和违约后果被串成一条明显偏向采购方的责任链，尤其是财政审批延期免责、争议款项暂停支付、第三方检测不合格即单方解除、送检和整改费用全由供应商承担等条款，明显加重供应商风险。
- 修改建议：拆分付款、验收、复检、售后和违约条款，分别明确触发条件、证据标准和费用边界；删除开放式免责和当然由供应商承担全部费用的表述。

### 7. 与医院信息化和碳足迹相关的附加义务外扩

- 位置：第三章 用户需求书-其他
- 页码提示：第42页左右
- 条款编号：9.5、9.9
- 辅助行号：1870-1881
- 原文摘录：
  - `免费开放软件端口...无偿派人配合与医院信息系统（包括但不限于HIS、PACS、LIS等系统）的连接工作`
  - `合同签订后30个日历天内...提供碳足迹盘查报告`
- 风险类型：`template_mismatch` / `mixed_scope_boundary`
- 风险等级：中高
- 合规判断：`potentially_problematic`
- 依据：
  - 《政府采购需求管理办法》
- 适用逻辑：血透类设备采购中确有信息系统对接可能，但当前条款把软件端口开放、无偿配合医院信息系统对接、碳足迹盘查和持续改进等义务一并压给供应商，边界明显扩大，且缺少与本项目范围和预算对应的论证。
- 修改建议：将确属设备联机和数据交换的必要接口要求单独列明；删除与本项目无直接关系的碳足迹盘查和持续监控义务，或转为采购人自行管理事项。

## 三、建议的总体整改方向

- 资格条件仅保留医疗器械法定许可、主体资格和与血透设备供货直接相关的最低履约条件。
- 重构评分表，删除与履约弱相关的荣誉、物业类证书、资本/营收/净利润加分，压降业绩和认证权重。
- 对技术参数中的第三方检测、标准引用、平台截图要求逐项做必要性论证，无法论证的删除或改成等效证明。
- 将售后、付款、验收、复检、违约条款拆开写清边界，避免“一切费用由供应商承担”“采购人不承担违约责任”等绝对化表述。
- 信息化接口和碳足迹相关义务应与设备采购范围严格对齐，超出部分删除或另行立项。

## 四、需要重点人工复核的事项

- 血透设备售后响应时限是否有医院临床安全管理的专门依据。
- 技术参数中的第三方检测、EN14175-3、ISO 20743、ISO 10993 等要求是否确与所购设备核心性能直接相关。
- 医院信息系统接口开放和数据对接是否属于本项目预算和范围内的必需工作。
- 碳足迹盘查要求是否属于本项目必要履约内容，还是模板残留。
