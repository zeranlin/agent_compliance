# 正式采购需求正文边界过滤层设计

## 背景

当前代码审查已经具备：
- 品目识别与场景理解
- 规则扫描
- 主题分析器
- 大模型关键节点混合介入
- 仲裁、法规语义与证据选择

但在真实文件里仍出现一类严重误报：
- 将 `特别警示条款`
- `投标文件组成要求及格式`
- `政府采购投标及履约承诺函`
- 平台操作提示、制作工具提示、通用格式说明

误当成正式采购需求正文的一部分，并进一步上浮为主问题。

这类误报不是“规则再补一条”能解决的问题，而是主链缺少一层明确的：
- `effective_requirement_scope_filter`

也就是先判断一段内容是不是“当前应作为采购需求风险识别对象的正式正文”。

## 目标

第一版目标很收敛：
- 把文档内容先分成 `正文 / 模板 / 提示 / 格式`
- 只让 `正文` 高权重进入主风险判断
- 让模板、提示、格式文本不再污染：
  - `review.py` 的初始 finding 生成
  - 混合边界和模板域主题上浮
  - `finding_arbiter` 的最终结果
  - `evidence_selector` 的代表性证据摘录

## 分类口径

### 1. 正文 `body`

指真正属于采购需求合规审查对象的内容，例如：
- 资格条件
- 评分标准
- 技术/服务要求
- 商务条款
- 验收要求
- 合同条款

### 2. 模板 `template`

指投标人填报、签署或承诺用的模板材料，例如：
- 投标函
- 承诺函
- 声明函
- 授权委托书
- 法定代表人证明书

### 3. 提示 `hint`

指警示、平台操作、制作工具或非需求实质说明，例如：
- 特别警示条款
- 温馨提示
- 文件创建标识码
- 投标文件制作工具说明
- 平台风险知悉文本

### 4. 格式 `format`

指投标文件编制说明、格式说明和信息公开分区说明，例如：
- 投标文件组成要求及格式
- 填写说明
- 格式自定
- 信息公开部分/不公开部分说明

## 主链接入点

### A. `review.py`

第一层过滤。

接入点：
- 初始 `RuleHit -> Finding` 组装前
- 主题 analyzer 收集候选 clause 时
- `_build_theme_finding()` 最终组装前

目标：
- 先把明显不是正式正文的命中排掉
- 防止模板/提示/格式文本进入主问题生成

### B. `finding_arbiter`

第二层过滤。

接入点：
- `apply_finding_arbiter()` 入口

目标：
- 即使上游仍有漏网 finding，也在最终仲裁前再做一次正文边界清理

### C. `evidence_selector`

第三层过滤。

接入点：
- `select_representative_evidence()`

目标：
- 当同一 finding 的 `source_text` 同时混有正文和提示/模板片段时
- 优先摘取正式正文片段
- 避免最终输出把提示文本当作代表性证据

## 第一版实现原则

第一版先采用轻量启发式，不引入复杂模型分类：
- 基于 `section_path`
- `source_section`
- `table_or_item_label`
- `clause_id`
- `text`

做规则化分类。

优先级：
1. `hint`
2. `format`
3. `template`
4. `body`

原因：
- 提示性文本最容易污染
- 格式说明和模板也不应优先进入正文风险判断
- 未命中上述标记时，保守按 `body` 处理

## 当前第一版边界

第一版明确不追求一次解决所有文档边界问题。

当前重点先压这类高频污染：
- 平台警示和工具提示
- 投标文件格式说明
- 承诺函/声明函模板
- 信息公开分区说明

暂未完全解决的情况：
- 某些正文和格式说明混排的表格
- 某些跨页标题残缺
- 某些既像模板又像正文的特殊本地化文件

这些留给二阶段继续增强。

## 二阶段方向

后续可以继续增强为：
- `body_primary / body_secondary / non_body` 三层权重
- 与 `procurement_stage_router` 联动
- 与 `catalog_knowledge_profile` 联动
- 对 `LLM` 输入候选段也做同样过滤

## 预期收益

补上这层后，预期直接改善：
- 混合边界误报
- 模板错贴主题误报
- 代表性证据错误引用
- 章节主问题被提示文本带偏

一句话说：

这层的作用不是多找问题，而是先回答：
**哪些内容才是真正应该被审查的采购需求正文。**
